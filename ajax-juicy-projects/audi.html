Project Description
The project involved a minimalistic design specific micro site for the launch of newly launched Audi R8 and A4 in Singapore. It involved a mobile friendly responsive site built using html, javascript, jquery and twitter bootstrap. It was delivered in a short notice for the motorshow where the cars were launched.
